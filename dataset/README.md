# dataset

The directory is used to save $Dataset_{MLM}$.

1. Download [mlm4dnn_dataset.tar.gz](https://mega.nz/file/hzVRiRAD#dXhIlq8J2uMJcMTcgwRjf3_MyTIcFxC9Wo71HZIQSf8)
2. Extract and move it here: `tar xzvf mlm4dnn_dataset.tar.gz`

The tree of this folder is as follows:
```
.
├── README.md
└── mlm4dnn_dataset
    ├── train.jsonl
    ├── valid.jsonl
    └── test.jsonl
```
